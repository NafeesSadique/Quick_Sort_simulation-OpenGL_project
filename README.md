# Quick Sort simulation using OpenGL


### Controls

>make sure Caps Lock is turned off

| Operations | keys |
| ------ | ------ |
|start/stop sorting|Enter|
|speed up sorting|m|
|slow down sorting|n|
|start/stop fan|left click (mouse)|
| ------ | ------ |
|move forward|w|
|move backward|s|
|move right|d|
|move left|a|
| ------ | ------ |
|look up|up arrow|
|look down|down arrow|
|look right|right arrow|
|look left|left arrow|
| ------ | ------ |
|front view|i|
|back view|k|
|right view|l|
|left view|j|
| ------ | ------ |
|lift up|spacebar|
|go down|c|
| ------ | ------ |
|exit|ESC|
